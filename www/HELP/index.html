General

REPRATE is a web app which allows you to store Beats which store the tempo for a exercise using the 4 point tempo system.And manage the beats.

If you are a weight lifter looking to maximise your results by using tempo.REPRATE is for you.

REPRATE was developed by Sandaru Adithya in 2019.On the back end,reprate uses Nodejs,ExpresJS,Mongodb.on the front end jQuery,Sweetalert,Vanilla JS and a few other technologies were used.

Please contack sandaruadithya@gmail.com.

Using REPRATE

 Pharse name  Description
 Eccentrick contraction  (Negetive rep)
 Strch position  (Pause at the end of the rep)
 Concentrick contraction  (Push/pull)
 Contracted position  (Pause at the upper point A.K.A - Squeeze)

Click in + button in the tool bar(top).

go to Home/My beats,click on the beat to detete,click delete button.

goto home,click the beat you want to play,press play button.

Click on the beat to edit,click edit icon in the player.

 Phase  Voice instruction
 Eccentrick contraction  Down
 Strch position  Strech
 Concentrick contraction  Up
 Contracted position  squeeze
